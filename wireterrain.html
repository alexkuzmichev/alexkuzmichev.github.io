Wire Terrain

Buy in Asset Store WebGL demo Documentation
Wire terrain package allows you procedurally create wireframe meshes from Unity Terrain objects. Generated meshes are native Unity meshes that are created with flag “MeshTopology.Lines”. These meshes are created in 3d space. There are two types of wire representation that you can create from terrain:
— Contours
— Grid
Unity forum thread: https://forum.unity.com/threads/released-wire-terrain.520407/
